# ECG - Intelligent Arrhythmia Monitor

Embedded ECG monitoring system using:

- ESP32-S3
- AD8232
- Pan-Tompkins algorithm
- MIT-BIH Arrhythmia Database
- CNN classification

## Features

- Real-time ECG acquisition
- R peak detection
- BPM estimation
- Arrhythmia classification
- Embedded signal processing

## Hardware

- ESP32-S3 DevKitC-1
- AD8232 ECG module

## Software

- Arduino Framework / ESP-IDF
- Python
- TensorFlow
- WFDB
- Scipy

## Dataset

MIT-BIH Arrhythmia Database
